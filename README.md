# 🎓 Student Performance Predictor

## 📌 Description
This project predicts student exam marks using Machine Learning and provides study improvement suggestions.

## 🚀 Features
- Predicts marks based on:
  - Study hours
  - Attendance
  - Sleep hours
  - Previous marks
- Gives personalized suggestions

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run

1. Install dependencies:
pip install pandas
pip install scikit-learn

2. Run the program

3. Enter values when prompted

## 📊 Example Output
Predicted Marks: 68
Performance: Good

Suggestions:
- Increase study hours
- Improve attendance
